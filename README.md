# So Meta

So Meta is a gem to simply manage meta content (title, description, canonical url, etc.) from within a Rails application

## Installation

Add this line to your application's Gemfile:

    gem 'so_meta'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install so_meta
